# IOUGURT STARTUP

## Descrição do Projeto 

Este projeto visa desenvolver uma plataforma web centralizada para clínicas veterinárias, permitindo o gerenciamento eficiente de pacientes, tutores, agendamentos e histórico clínico. O sistema foi concebido para otimizar as operações diárias da clínica, proporcionando uma interface intuitiva para os profissionais.

### Objetivo da Plataforma

O principal objetivo da plataforma é oferecer uma ferramenta completa para clínicas veterinárias gerenciarem seus processos internos. Isso inclui desde o cadastro detalhado de pacientes e seus tutores até o acompanhamento do histórico de atendimentos e a organização da agenda de consultas e procedimentos.
